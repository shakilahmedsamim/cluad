# Services

This file documents the services offered. To update service descriptions, edit `src/data/content.js`.

## Available Services

- Google Tag Manager Setup
- GA4 Tracking Setup  
- Server-Side Tracking
- Meta Conversion API
- Google Ads Conversion Tracking
- Enhanced Conversions
- Tracking Audit
- Revenue Attribution
- Offline Conversion Tracking
- First-Party Tracking

To add or edit services, update the `services` array in `/src/data/content.js`.
